考试模式
---------------
考试科目-自写 <br>
添加部分-考试多少个部分加多少部分<br>
（请写好时间，以分钟为单位）<br>
之后按“开始考试” 计时器就会运行<br>
<img width="1832" height="915" alt="image" src="https://github.com/user-attachments/assets/c3529b72-295a-49bd-bd84-90c1e1bd17b8" />


普通倒计时
----------------
写上自己需要的时间即可，以分钟为单位 <br>
内设开始，暂停，重置按键<br>
按“开始”即可运行<br>
如需要停止可以按“暂停” <br>
重置请按“重置” <br>

To-Do
-------------
由于目前还没增加日期功能入内，只好让大家自己先在自己的内容前或后标上日期 <br>
已分好“待办”和“已完成”的部分 <br>
做完的To-Do可以在旁边按 “✔”标志以表示完成，之后会自动进入“已完成” <br>
已完成部分可以选择删除内容 <br>
此部分有自动存档，不必担心日后To-Do会消失 <br>
